KYC/AML Frontend Server

Quick Start

To serve the KYC frontend, run one of these commands:

Python 3 (Recommended)

python -m http.server 8080

Then open: http://localhost:8080/frontend/

Node.js (if installed)

npx http-server -p 8080

Then open: http://localhost:8080/frontend/

Python 2

python -m SimpleHTTPServer 8080

Then open: http://localhost:8080/frontend/

API Gateway Status

Ensure the API Gateway is running on port 8000

Next Steps

  1. Start Docker: docker-compose up -d
  2. Start frontend server: python -m http.server 8080
  3. Open frontend: http://localhost:8080/frontend/
  4. Upload a document and test!
